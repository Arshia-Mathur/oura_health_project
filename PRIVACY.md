Privacy Policy

Last updated: July 2026

This privacy policy applies to a personal, single-user application ("the App") built to connect one individual's Oura Ring account to a local, personal health-tracking tool. The App is not a commercial product, is not distributed to other users, and is not intended for use by anyone other than its creator.

What data is accessed

With authorization through Oura's OAuth2 flow, the App accesses the following data types from the Oura Cloud API: personal profile information (age, gender, height, weight), daily activity, workouts, sleep, readiness, heart rate, and related wellness metrics, depending on the scopes granted at authorization time.

How the data is used

Data is used exclusively for personal tracking and analysis: generating daily progress summaries, checking activity and sleep trends against personal fitness goals, and producing periodic notifications to the account owner. Data may be processed locally by AI tools (Claude Code, by Anthropic) to generate these summaries and insights. This processing happens as part of the owner's own use of their own data — no data from this App is shared with, sold to, or made accessible to any other individual or organization.

Where the data is stored

All synced data is stored locally on the account owner's own device, in a local database file with restricted (owner-only) file permissions. OAuth tokens are also stored locally, are never transmitted to any party other than Oura's API for authentication, and are never committed to any public code repository.

No data collected through this App is stored on any third-party server, sold, shared with advertisers, or used for any purpose beyond the personal tracking described above.

Data retention and deletion

Since all data is stored locally, the account owner may delete it at any time by removing the local database and token files. OAuth access can be revoked at any time through Oura's own account settings (cloud.ouraring.com), which immediately stops the App's access to Oura data.

Third parties

This App is not affiliated with, endorsed by, or sponsored by Ōura Health Oy. It is an independent, personal integration built against Oura's public developer API.
